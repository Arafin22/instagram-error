What is js event loop do?

Event loop js এর instruction গুলকে executen করে |js এর synchronous and asynchronous program গুলাকে কোনটার পর কোনটার চলবে সেটা সিদ্ধান্ত নেয় |js এর single thread টা event looop ভিতরে থাকে। js engine এর ভিতরে Heap ও call stack থাকে | call stack js program function call গুলা stack আকারে থাকে(last in first out) | একটা করে function execute হলে stack থেকে বের হয়েযায় | যতখন stack function call থাকে তওখন event loop চলতে থাকবে |


Difference localstorage sessionstorage?

Localstorage:Browser tab close করলে ও expire হয়না।এটার মেমোরি browser dependent(অনেক বেশী)।এটার accessibility all browser tab এ |

Sessionstorage:Browser tab close করলে expire হয়।এটার মেমোরি 5mb ।এটার accessibility only present tab এ |
